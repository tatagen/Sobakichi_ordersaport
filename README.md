# そばきち オーダーサポートアプリ

飲食店のフロアスタッフ向け研修ツールです。メニューを選択すると、調理手順・伝票記入フォーマット・提供セットが表示されます。

## 概要

新人スタッフがオーダーの取り方・提供内容を素早く確認できるよう設計した、シンプルなWebアプリです。外部ライブラリに依存しない軽量設計で、スマートフォンでも使いやすいレスポンシブUIを採用しています。

## 機能

- メニュー別の調理指示の表示
- 伝票記入フォーマットの確認
- 薬味・セット内容の確認
- スマートフォン対応（max-width: 480px）

## 技術スタック

| カテゴリ | 技術 |
|---|---|
| フロントエンド | HTML / CSS / Vanilla JavaScript |
| データ管理 | JSON ファイル |
| アーキテクチャ | クライアントサイドのみ（外部ライブラリ不使用） |

## 動作方法

`fetch()` でJSONデータを読み込むため、ローカルサーバーが必要です。

### VS Code Live Server を使う場合
`index.html` を右クリック → `Open with Live Server`

### Python を使う場合
```bash
python -m http.server 8080
# ブラウザで http://localhost:8080 を開く
```

### Node.js を使う場合
```bash
npx serve .
```
